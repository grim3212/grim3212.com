All armor materials added by GrimPack allow for configuration. You can only configure this by finding your config folder, inside the grimpack folder, is a materials folder, and then any thing that ends in armor_material. 

{
	"durability": 35,
	"reductionAmounts": [
		4,
		8,
		10,
		4
	],
	"enchantability": 20,
	"toughness": 2.5
}

All values for armor material can be modified.

  • durability refers to how long the armor will last. This value is multiplied differently for each piece.
    • durability * 11 for the helmet
    • durability * 16 for the chestplate
    • durability * 15 for the leggings
    • durability * 13 for the boots
  • reductionAmounts refers to the number of armor points the player gets for wearing each piece of armor. Follows the same pattern of first helmet, then chestplate, leggings, and finally boots.
  • enchantibility refers to how enchantible the material is. The higher the number the better enchantments are likely to appear.
