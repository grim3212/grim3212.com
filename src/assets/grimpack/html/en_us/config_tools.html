All tool materials added by GrimPack allow for configuration of everything. You can only configure this by finding your config folder, inside the grimpack folder, is a materials folder, and then any thing that ends in tool_material. 

{
	"harvestLevel": 4,
	"maxUses": 5122,
	"efficiency": 15.0,
	"damage": 5.0,
	"enchantability": 20
}

As you can see all values for each tool material may be modified.
  • harvestLevel for tools, refers to what blocks it can break (a higher level is higher tier blocks).
  • maxUses refers to how many uses it has before it breaks.
  • efficiency for tools, refers to how quickly the material can break blocks.
  • damage refers to damage to entities.
  • enchantability refers to how enchantible the material is. The higher the number the better enchantments are likely to appear.
