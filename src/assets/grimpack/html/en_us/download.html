GrimPack is built with Minecraft Forge so the first thing to do would be to install that. Then you would need the mod itself.

You can get it from Curse Forge, if you got it from somewhere else it is not official. You should immediately remove it and download the official version.

Once you have installed Forge simply drop the GrimPack-MCVersion-ModVersion.jar into your mods folder and you are ready to go.

Requirements:
  • Minecraft Forge
  • Grim Pack
There are a few optional mods that GrimPack has functionality with. There is Just Enough Items (JEI) and Just Enough Resources (JER).
GrimPack has custom recipe handlers with JEI for things like the Grill and Machines. It also has loot handlers with JER.

Optional Mods:
  • Just Enough Items
  • Just Enough Resources
