GrimPack uses the json recipe format for almost every recipe except for coloring Backpacks and coloring Pellet Bags.

Included along with GrimPack recipes are many condition's that you can check for.
  • ore_dict condition which checks if 'oreName' exists
  • part_enabled condition checks if the 'part' is loaded
  • (core | cuisine | decor | industry | tools | util | world)_subpart_enabled checks to see if the part is loaded and if it loaded with the 'subpart' provided


Finally there are a few custom recipe formats that are included Grills, machine, and chisel.
Chisel: 

{
"type": "grimpack:chisel",
"conditions": [
    {
      "type": "grimpack:tools_subpart_enabled",
      "subpart": "chisel"
    }
  ],
  "inBlock": {
    "item": "minecraft:iron_ore"
  },
  "outBlock": {
    "item": "minecraft:stone",
    "data": 0
  },
  "outItems": {
    "items": [
    	{
    		"item": "grimpack:iron_ore_item"
    	}
    ]
  }
}
  • inBlock is the block that the chisel will change
  • outBlock is the block that the inBlock will transform into
  • outItems are the items that will be given to the player
  • defaultDrops if true will give the player the default drops if the block was broken


Grills: 

{
"conditions": [
    {
      "type": "grimpack:decor_subpart_enabled",
      "subpart": "fireplaces"
    }
  ],
  "type": "grimpack:grill",
  "input": {
  	"item": "minecraft:chicken"
  },
  "output": {
    "item": "minecraft:cooked_chicken"
  },
  "experience": 0.35
}
  • input is the item to grill
  • output is the item after it has been fully grilled
  • experience is the amount of experience to give the player


All machines follow the same pattern of
  • input for input item
  • output for item returned
  • experience for xp to give to the player
  • machine to specify which type of machine it is (derrick | refinery | modern_furnace)

All machines also allow for the input to be an oredict name so all inputs of that orename will be changed into the output.

Derrick: 

{
"conditions": [
    {
      "type": "grimpack:industry_subpart_enabled",
      "subpart": "machines"
    },
    {
      "type": "grimpack:ore_dict",
      "oreName": "can"
    }
  ],
  "type": "grimpack:machine",
  "machine": "derrick",
  "input": {
  	"item": "#CAN"
  },
  "output": {
    "item": "grimpack:crude_oil"
  },
  "experience": 0.1
}
